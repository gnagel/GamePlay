int gameplay::Platform::enterMessagePump ( )

Begins processing the platform messages.

This method handles all OS window messages and drives the game loop. It normally does not return until the application is closed.

If a attachToWindow is passed to Platform::create the message pump will instead attach to or allow the attachToWindow to drive the game loop on the platform.

static void gameplay::Platform::setMouseCaptured ( bool  captured)
static

Enables or disabled mouse capture.

When mouse capture is enabled, the platform cursor is hidden and mouse event points are delivered as position deltas instead of absolute positions.

This is useful for games that wish to provide uninhibited mouse movement, such as when implementing free/mouse look in an FPS game.

Disabling mouse capture moves the mouse back to the center of the screen and shows the platform cursor.

Note that this method does nothing on platforms that do not support a mouse.

Parameters
capturedTrue to enable mouse capture, false to disable it.
